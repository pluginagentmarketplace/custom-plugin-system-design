# /skill-assessment

Evaluate your current skills and get personalized recommendations for learning paths.

## Usage

```
/skill-assessment
/skill-assessment [skill-name]
/skill-assessment [role]
```

## How Assessment Works

### Step 1: Select Domain
Choose the skill or role you want to assess:
```
/skill-assessment

> Select assessment:
  1. Frontend Development
  2. Backend Development
  3. Mobile Development
  ... (continue through all 7)
```

### Step 2: Answer Questions
Interactive questions tailored to your domain:
- Multiple choice
- Free-form coding
- Design questions
- Scenario analysis

### Step 3: Get Results
Detailed report including:
- **Overall Score** (0-100%)
- **Category Breakdown** - Strengths and weaknesses
- **Level Assessment** - Beginner/Intermediate/Advanced/Expert
- **Skill Gap Analysis** - What to learn next
- **Personalized Recommendations** - Specific learning paths

## Understanding Your Results

### Score Ranges
- **90-100%** - Expert level, consider mentoring others
- **80-89%** - Advanced, ready for complex projects
- **70-79%** - Good intermediate level
- **60-69%** - Foundational but gaps exist
- **Below 60%** - Focus on fundamentals

### What Next?

**For High Scores (80%+):**
- Explore advanced topics
- Mentor others
- Take on architectural responsibilities
- Consider specialized paths

**For Moderate Scores (60-79%):**
- Follow personalized learning path
- Build projects to reinforce learning
- Practice with code challenges
- Take advanced courses

**For Lower Scores (Below 60%):**
- Start with fundamentals
- Use structured learning paths
- Build small projects
- Get mentorship

## Retake Assessment

- Use results to adjust learning path

## Pro Tips

1. **Be Honest** - Assessment only works if you answer truthfully
2. **Relax** - This is for your growth, not grading
3. **Retry Weak Areas** - Come back to questions you weren't sure about
4. **Action Plan** - Use results to create concrete learning goals
5. **Track Progress** - Retake periodically to see improvement
