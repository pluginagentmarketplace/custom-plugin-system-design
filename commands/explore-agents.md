# /explore-agents

Explore all 7 specialized agents and their capabilities for discovering expert guidance.

## Usage

```
/explore-agents
/explore-agents [agent-name]
```

- Frontend performance optimization

**When to Ask:**
- Building interactive web applications

## How to Use Agents

### 1. Get Agent Overview
```
/explore-agents

Shows all 7 agents with brief descriptions
```

### 2. Learn About Specific Agent
```
/explore-agents backend-server-development

Detailed capabilities, learning paths, and when to use
```

### 3. Agent Interaction
Once you select an agent, you can:
- Ask for guidance on specific topics
- Request code examples
- Get career advice
- Discuss complex architectural decisions
- Get mentorship on best practices

## Agent Interaction Examples

**Asking the Backend Agent:**
> "I need to design a payment processing system. What's the best approach?"

**Asking the DevOps Agent:**
> "We're moving from monolithic to microservices. How do we set up Kubernetes?"

**Asking the Security Agent:**
> "What's the best way to store API keys in our application?"

**Asking the Leadership Agent:**
> "How do I manage a team of 10 engineers with varying skill levels?"
